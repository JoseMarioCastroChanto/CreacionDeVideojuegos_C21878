\doxysubsection{Detailed Description}
\doxylink{classSystem}{System} responsible for updating the animation frames of entities. 

This system requires entities to have both \doxylink{structAnimationComponent}{Animation\+Component} and \doxylink{structSpriteComponent}{Sprite\+Component}. It updates the current animation frame based on time and adjusts the source rectangle of the sprite accordingly.
