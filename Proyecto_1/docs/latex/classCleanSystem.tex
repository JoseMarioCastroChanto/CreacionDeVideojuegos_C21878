\doxysubsection{Detailed Description}
\doxylink{classSystem}{System} responsible for removing entities that move outside defined bounds. 

This system requires entities to have both \doxylink{structRigidBodyComponent}{Rigid\+Body\+Component} and \doxylink{structTransformComponent}{Transform\+Component}. It checks the x position of the entity\textquotesingle{}s transform, and if it is outside the range \mbox{[}-\/400, 2000\mbox{]}, it kills the entity and logs the removal.
