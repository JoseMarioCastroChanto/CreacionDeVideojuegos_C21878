\doxysubsection{Detailed Description}
Checks if any objective entity is defeated (life \texorpdfstring{$<$}{<}= 0). 

If any objective entity without the \doxylink{structTagProjectileComponent}{Tag\+Projectile\+Component} reaches zero or less life, the Defeat flag is set to true.
