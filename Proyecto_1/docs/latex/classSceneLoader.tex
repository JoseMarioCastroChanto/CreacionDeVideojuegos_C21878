\doxysubsection{Detailed Description}
Loads a game scene from a Lua script. 

This class is responsible for parsing Lua scripts to load game assets, entities, key and button bindings, and music into the appropriate managers and systems (\doxylink{classAssetManager}{Asset\+Manager}, \doxylink{classControllerManager}{Controller\+Manager}, \doxylink{classRegistry}{Registry}, etc.).
