\doxysubsection{Detailed Description}
Handles collisions between wall entities and player entities. 

This system requires entities with the \doxylink{structTagWallComponent}{Tag\+Wall\+Component}. It listens for collision events and when a player collides with a wall, it adjusts the player\textquotesingle{}s position to prevent overlapping and stops their movement by zeroing velocity.
