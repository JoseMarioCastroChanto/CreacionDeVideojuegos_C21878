Updates and renders all text entities. 

For each entity\+:
\begin{DoxyItemize}
\item Renders the text surface with the font and color specified in \doxylink{structTextComponent}{Text\+Component}.
\item Creates a texture from the surface.
\item Uses the \doxylink{structTransformComponent}{Transform\+Component} for positioning and scaling.
\item Renders the texture on the screen.
\item Frees the texture and surface after rendering.
\end{DoxyItemize}
