\doxysubsection{Detailed Description}
\doxylink{classSystem}{System} responsible for updating entity positions based on their velocity. 

This system requires entities to have both \doxylink{structRigidBodyComponent}{Rigid\+Body\+Component} and \doxylink{structTransformComponent}{Transform\+Component}. It updates the position of each entity every frame according to its velocity and the elapsed time.
