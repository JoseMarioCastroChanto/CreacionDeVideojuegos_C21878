\doxysubsection{Detailed Description}
\doxylink{classSystem}{System} responsible for detecting collisions between entities and emitting collision events. 

This system requires entities to have a \doxylink{structTransformComponent}{Transform\+Component}. It supports collision detection between entities with \doxylink{structCircleColliderComponent}{Circle\+Collider\+Component} and \doxylink{structPolygonColliderComponent}{Polygon\+Collider\+Component}.

The system detects collisions using circle-\/vs-\/circle and circle-\/vs-\/polygon algorithms. Upon detecting a collision, it emits a \doxylink{classCollisionEvent}{Collision\+Event} through the \doxylink{classEventManager}{Event\+Manager}.
