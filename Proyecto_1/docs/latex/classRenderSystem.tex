\doxysubsection{Detailed Description}
\doxylink{classSystem}{System} responsible for rendering entities with sprites on the screen. 

This system requires entities to have \doxylink{structSpriteComponent}{Sprite\+Component} and \doxylink{structTransformComponent}{Transform\+Component}. It sorts entities based on their \doxylink{structDepthComponent}{Depth\+Component} (if present) to render in the correct order.
