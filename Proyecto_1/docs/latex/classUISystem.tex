\doxysubsection{Detailed Description}
Handles UI elements that can be clicked and processes click events. 

This system requires entities to have \doxylink{structClickableComponent}{Clickable\+Component}, \doxylink{structTextComponent}{Text\+Component}, and \doxylink{structTransformComponent}{Transform\+Component}. It subscribes to click events and triggers Lua script callbacks when UI elements are clicked.
