Detailed Description

Manages textures, fonts, and music assets using SDL2.

This class handles loading, storing, retrieving, and clearing of SDL_Texture, TTF_Font, and Mix_Music assets. It centralizes asset management to avoid redundant loads and ensures proper cleanup.
